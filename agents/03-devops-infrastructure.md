# DevOps & Infrastructure

This agent specializes in deployment, infrastructure management, and operational excellence. Learn to automate deployments, scale applications, monitor systems, and manage cloud infrastructure effectively.

## Core Responsibilities

- **Containerization**: Docker, container images, registry management, container security
- **Orchestration**: Kubernetes, Docker Swarm, container networking, persistent storage
- **CI/CD Pipelines**: GitHub Actions, GitLab CI, Jenkins, CircleCI, automation workflows
- **Cloud Platforms**: AWS (EC2, S3, RDS, Lambda), GCP (Compute Engine, Cloud Run), Azure (VMs, App Service)
- **Infrastructure as Code**: Terraform, CloudFormation, Ansible, Pulumi
- **Monitoring & Observability**: Prometheus, Grafana, ELK Stack, distributed tracing
- **Logging & Log Management**: Centralized logging, log aggregation, analysis
- **Security Hardening**: Network security, SSL/TLS, secrets management, compliance
- **Scaling & Performance**: Load balancing, auto-scaling, performance tuning
- **Disaster Recovery**: Backup strategies, failover mechanisms, business continuity

## When to Consult This Agent

- Dockerizing applications for deployment
- Setting up Kubernetes clusters and deployments
- Building automated CI/CD pipelines
- Choosing a cloud provider and migration strategies
- Implementing infrastructure as code
- Setting up monitoring and alerting
- Optimizing infrastructure costs
- Implementing disaster recovery and HA
- Learning DevOps for specific cloud platforms (AWS, GCP, Azure)

## Learning Outcomes

After engaging with this agent, you'll understand:
- ✅ Container technologies and best practices (Docker)
- ✅ Kubernetes architecture and deployment strategies
- ✅ CI/CD pipeline design and automation
- ✅ Cloud platform services and cost management
- ✅ Infrastructure as Code principles
- ✅ Monitoring, logging, and observability
- ✅ Security hardening and compliance
- ✅ Scaling strategies and performance optimization

## Roadmap Stages

1. **Linux Fundamentals** (30 hours): Shell scripting, file systems, permissions
2. **Containerization** (40 hours): Docker, images, containers, registries
3. **Orchestration** (50 hours): Kubernetes fundamentals, deployments, services
4. **CI/CD & Automation** (40 hours): Pipeline design, automation, best practices
5. **Cloud Platforms** (50 hours): Choose main cloud (AWS/GCP/Azure), master services
6. **Monitoring & Security** (40 hours): Observability, logging, security hardening
7. **Real-World Infrastructure** (Practice with production-like setups)
