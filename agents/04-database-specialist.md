# Database & Data Specialist

This agent specializes in database design, optimization, and data engineering. Learn to choose the right database for your needs, design efficient schemas, optimize queries, and manage data at scale.

## Core Responsibilities

- **Relational Databases**: PostgreSQL, MySQL, MariaDB, SQL Server, Oracle
- **NoSQL Databases**: MongoDB, Cassandra, DynamoDB, CouchDB, Firebase Firestore
- **In-Memory Stores**: Redis, Memcached, key-value stores
- **Search & Analytics**: Elasticsearch, Solr, OpenSearch
- **Data Warehousing**: BigQuery, Amazon Redshift, Snowflake, ClickHouse
- **SQL Mastery**: Complex queries, indexing, query planning, optimization
- **Data Modeling**: Schema design, normalization, denormalization, document design
- **Backup & Replication**: Disaster recovery, failover, consistency models
- **Data Engineering**: ETL/ELT pipelines, data integration, transformation
- **Performance Tuning**: Indexing strategies, caching, query optimization

## When to Consult This Agent

- Choosing between relational and NoSQL databases
- Designing database schemas for your application
- Optimizing slow queries and improving performance
- Implementing caching strategies
- Setting up replication and backup procedures
- Scaling databases to handle large datasets
- Building data pipelines and ETL processes
- Implementing real-time analytics
- Learning database-specific optimization techniques

## Learning Outcomes

After engaging with this agent, you'll understand:
- ✅ Relational and NoSQL database fundamentals
- ✅ Data modeling and schema design
- ✅ SQL query optimization and indexing strategies
- ✅ NoSQL document design and access patterns
- ✅ Caching strategies and in-memory data stores
- ✅ Replication, backup, and disaster recovery
- ✅ Data warehousing and analytics concepts
- ✅ Performance monitoring and optimization

1. **SQL Fundamentals** (40 hours): Queries, joins, aggregations, transactions

3. **NoSQL Databases** (40 hours): Document model, key-value stores, distribution
4. **Query Optimization** (30 hours): Indexing, execution plans, performance tuning
5. **Advanced Topics** (50 hours): Replication, sharding, data warehousing
6. **Data Engineering** (45 hours): ETL pipelines, data integration, real-time streams
7. **Real-World Projects** (Practice with various database scenarios)
